# Documento Detallado de Diseño (DDD) - Página Web de Martin Jaimes

## Descripción del trabajo
Esta página web personal tiene como objetivo presentar información sobre mí, mi formación académica y mis intereses personales y profesionales. El sitio web está diseñado como un proyecto para la asignatura **Fundamentos para la Ingeniería**, bajo la supervisión del profesor **Héctor Molina**.  

### Estructura del sitio web
El sitio está organizado en varias secciones y páginas interconectadas, siguiendo buenas prácticas de organización y diseño web:

1. **Inicio (index.html)**  
   - Página principal de bienvenida.  
   - Contiene información general sobre el sitio y su objetivo.  
   - Presenta imágenes, título, breve explicación del contenido de la web y un menú de navegación a las demás secciones.  
   - Incluye un FAQ con acordeón para preguntas frecuentes.

2. **Temática (paginas/topic.html)**  
   - Proporciona información completa sobre un tema de interés seleccionado por el autor.  
   - Permite demostrar la capacidad de organizar y mostrar contenido temático de forma clara.

3. **Grado (paginas/degree.html)**  
   - Describe mi grado académico y su propósito.  
   - Presenta la estructura completa del programa y lista de materias.  
   - Contiene una subpágina sobre **Fundamentos de la Ingeniería Informática (fii.html)**, donde se profundiza en objetivos, contenido y resultados de aprendizaje.

4. **Acerca de mí (paginas/about.html)**  
   - Información personal inspirada en los contenidos de las unidades 2 y 3 del módulo 1.  
   - Incluye currículum vitae en PDF, dos imágenes y enlaces a redes sociales.  

5. **Red (paginas/net.html)**  
   - Sección dedicada a la red de compañeros.  
   - Contiene nombres y enlaces a los sitios web de los compañeros, estableciendo conexiones académicas y colaborativas.

6. **Contacto (paginas/contact.html)**  
   - Formulario simple para ponerse en contacto conmigo.  
   - Diseñado sin elementos de script que envíen o guarden el contenido, cumpliendo con el requerimiento de seguridad del profesor.

### Hojas de estilo (CSS)
- Todos los estilos están centralizados en un archivo externo `CSS/estilo.css`.  
- Se utiliza **Bootstrap 5.3** para facilitar la creación de layouts responsivos y elementos interactivos, combinando HTML y CSS con clases predefinidas.

---

## Problemas durante el desarrollo
- Ajuste de estilos CSS para todas las páginas y consistencia en tipografía y colores.  
- Organización de imágenes en la carpeta `imagenes` y asegurarse de que las rutas sean correctas.  
- Vinculación correcta de todas las páginas entre sí a través del menú de navegación y los enlaces del footer.  
- Integración de Bootstrap y sus scripts en todas las páginas para los componentes interactivos (como acordeones y navegación responsive).
